2. Técnicas implementadas

Resumen de los conceptos aplicados en esta práctica.

🔒
novalidate + JS

Validación completamente controlada por JavaScript, sin burbujas nativas del navegador.

⚡
Tiempo real (focusout / input)

Validación al salir del campo y limpieza del error al volver a escribir.

📋
FormData API

Object.fromEntries(new FormData(form)) para obtener todos los datos como objeto.

💾
Autoguardado sessionStorage

El borrador se guarda automáticamente y se puede restaurar al recargar la página.

📱
Máscara de teléfono

Formato (099) 999-9999 aplicado visualmente mientras el usuario escribe.

🔑
Fuerza de contraseña

Indicador visual con 4 niveles: Débil, Regular, Buena y Fuerte.

Log de Operaciones
